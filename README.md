# ms-worker
Projecto de Micro Serviços com objectivo de colocar em prática conhecimento envolvendo o desenvolvimento de MicroServiços.

### Ferramentas usadas:
1. Java;
2. Spring Boot;
3.  FeignRibbon;
4.  Hystrix;
5.  OAuth; 
6.  JWT;
7.  Discovery Server Eureka;
8.  API Gateway Zuul;
9.  Config Server; &
10.  Docker.
