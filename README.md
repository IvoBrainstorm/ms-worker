# ms-worker
Projecto de Micro Serviços com objectivo de colocar em prática conhecimento envolvendo o desenvolvimento de MicroServiços.

### Ferramentas usadas:
1. Java;
2. Spring Boot;
3.  FeignRibbon;
4.  Hystrix;
5.  OAuth; JWT;
6.  Discovery Server Eureka;
7.  API Gateway Zuul;
8.  Config Server; &
9.  Docker.
